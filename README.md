# ainft-factory-program-contracts

Template code for AINFT contracts.

Expected node version is >= 16.0.0.

Make a copy of the `.env.example` and rename it `.env`. Fill out the keys in the file.

## Install dependencies

```shell
$ yarn
```

## Compile Solidity

```shell
$ yarn compile
```

## Lint Solidity

```shell
$ yarn lint:sol
```

## Apply prettier

```shell
$ yarn prettier
```

## Run tests

```shell
$ yarn test
```

## Run deploy script

```shell
$ yarn deploy:goerli
```

## Run mint script

```shell
$ yarn mint:goerli
```
